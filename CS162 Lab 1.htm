CS162 Lab 1

1.       Write a function that prints a pattern:

void print(int numberOfRows, int numberOfCols, char charUsed);

example:  print(3, 4, ‘*’); will print the following pattern

****

****

****

2.       Now imagine each char you print has a number associated. The first one printed has 1 and the second one has 2 and so on. For the chars whose value is 3’s multiple, your program should print a ‘3’ instead.

The above example:  print(3, 4, ‘*’); now should print the following pattern

**3*

*3**

3**3

3.       Copy/paste the print function you implement in step 2 into a plain text file, lab1.txt (you can use notepad for this) and upload it into lab 1 dropbox. For submission please only copy/paste the print function, but it’s your responsibility to write driver code and test it thoroughly before uploading the text file.
