CS162 Lab 2

1.       Given the following struct declaration:

const int MAX_CHAR = 100;

struct Student

{

   char id[MAX_CHAR];

   float gpa;

};

Implement the following functions:

bool insertStudent(Student list[], int& sizeOfList, const Student& aStudent);

The above function inserts “aStudent” into “list” according to the order of id. “list” should be ordered by student id. The function returns true if the operation is successful and false if a student with the same id exists.

 

bool removeStudent(Student list[], int& sizeOfList, const char id[]);

The above function removes the student object with “id” from “list.” It returns true if the operation is successful and false if the student doesn’t exist. The list should be ordered by student id at all time.

 

2.       Write driver program and design test cases to test the functions.
